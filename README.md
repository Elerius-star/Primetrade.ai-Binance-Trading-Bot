A professional, production-ready trading bot for Binance Futures Testnet with both CLI and Web interfaces. Features real-time price tracking, order management, portfolio monitoring, and comprehensive error handling.

![Trading Bot Dashboard](https://via.placeholder.com/800x400?text=Trading+Bot+Dashboard)

## ✨ Features

### Core Trading Features
- ✅ **Market Orders** - Instant execution at current market price
- ✅ **Limit Orders** - Set specific price for order execution
- ✅ **Both Sides** - Support for BUY and SELL orders
- ✅ **Real-time Prices** - Live price updates every 5 seconds
- ✅ **Portfolio Tracking** - Monitor balances and P&L in real-time
- ✅ **Order History** - Track all past orders and executions

### Technical Features
- ✅ **Dual Interface** - CLI and Web GUI available
- ✅ **Mock Mode** - Test without real API keys
- ✅ **Comprehensive Logging** - All actions logged to files
- ✅ **Input Validation** - Robust validation for all user inputs
- ✅ **Error Handling** - Graceful handling of API and network errors
- ✅ **Modular Architecture** - Clean separation of concerns

### UI/UX Features
- ✅ **Professional Design** - Dark theme with purple/orange gradients
- ✅ **Responsive Layout** - Works on desktop, tablet, and mobile
- ✅ **Real-time Updates** - Live price ticker and balance updates
- ✅ **Toast Notifications** - Instant feedback on actions
- ✅ **Order Preview** - Review orders before submission
- ✅ **Activity Feed** - Recent orders displayed in dashboard
